Static portfolio site (no build tools). Edit text/links, open `index.html`, and deploy anywhere.

## How to run

- Open `index.html` directly in a browser, or
- Use a local server (recommended):

```powershell
cd "c:\Users\Hp\Desktop\Protfilo website"
python -m http.server 5500
```

Then open `http://localhost:5500`.

## Customize

- Update your **email/GitHub/LinkedIn** in `script.js` under `CONFIG`.
- Replace the avatar in `assets/profile-placeholder.svg` (or use your own image).
- Add your resume PDF as `assets/Akash-Kumar-Resume.pdf` (the button will automatically switch if missing).
- Update projects in the `Projects` section in `index.html` + links in `script.js`.
